MemoAI V2 设置窗口问题修复说明

问题描述：
- 打开设置窗口时报错：AttributeError: 'App'对象没有'translations'属性
- 设置窗口内容为空

问题原因：
代码中使用了错误的属性名self.translations，应该使用self.translation_manager.get_text()

修复内容：
1. 修复了设置窗口中所有使用错误属性名的地方
2. 将所有self.translations[self.current_language]["key"]替换为self.translation_manager.get_text("key", self.current_language)

修复文件：
- memoAI_V2_UI.py：修复了设置窗口相关的7处翻译文本获取

验证结果：
- 系统自检通过
- 模型加载成功
- 记忆文件正常
- 设置窗口现在可以正常打开并显示内容

使用方法：
直接运行start.py即可启动程序，点击设置按钮可以正常打开设置窗口。

启动命令：
Windows: python start.py
Linux/macOS: ./start.sh 或 python start.py
